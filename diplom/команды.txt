docker run --name dgraph-dev -d -p 8080:8080 -p 9080:9080 dgraph/standalone:latest

Это поднимет Dgraph локально:

8080 - HTTP
9080 - gRPC, обычно нужен Python-клиенту



Как проверить, что база запустилась

Проверь контейнер: docker ps

Посмотри логи:

docker logs dgraph-dev




Как останавливать и запускать потом

Остановить:

docker stop dgraph-dev

Запустить снова:

docker start dgraph-dev

Удалить контейнер:

docker rm -f dgraph-dev
